## 🔄 CUSTOMIZATION GUIDE

### Change Colors
Edit CSS variables in your globals.css:
```css
--logo-primary: your-color;
--logo-accent: your-color;
```

### Change Size
Use size prop:
```jsx
<WidgetLogoIcon size="lg" />
```

### Change Appearance
Use variant prop:
```jsx
<WidgetLogoIcon variant="minimal" />
```

### Add Animation
Use animate prop:
```jsx
<WidgetLogo animate={true} />
```

### Custom Classes
Use className prop:
```jsx
<WidgetLogoIcon className="custom-class" />
```
